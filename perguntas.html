Exercícios

Todos os exercícios solicitados pelo ChatGPT e utilizados nesse projeto de estudo

  • Crie uma lista ordenada com 3 itens: "Maçã", "Banana" e "Laranja".
  • Insira uma imagem com o arquivo "gato.jpg" e com o texto alternativo "Foto de um gato".
  • Crie um link que abra uma nova aba e leve o usuário para o site "https://www.google.com" com o texto "Ir para o Google".
  • Crie um formulário com um campo de texto para o nome e um botão de envio.
  • Crie uma página HTML simples com um título, um subtítulo e um parágrafo de texto.
  • Crie uma lista não ordenada com pelo menos 3 itens.
  • Adicione um link para outra página da web.
  • Insira uma imagem na página.
  • Crie um formulário com um campo de entrada de texto e um botão de envio.
  • Adicione uma tabela com pelo menos 3 colunas e 3 linhas.
  • Crie uma página de login com campos de entrada para nome de usuário e senha.
  • Crie uma página com pelo menos 3 seções diferentes usando tags de seção HTML, como section, article e aside.
  • Crie um menu de navegação usando a tag nav e links para outras páginas em seu site.
  • Crie um rodapé com informações sobre o seu site, como o ano de criação, o nome do proprietário e links para redes sociais.
